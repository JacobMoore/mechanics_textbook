Get Involved in the Mechanics Map Project

The Mechanics Map Open Textbook Project is an ongoing effort of lead author Jacob Moore, but many others have contributed to the project along the way, and I am always looking for others who want to help. If you are interested in contributing to the project, the first step would be to send me an email at jmoore@psu.edu.

Both technical and non-technical help is appreciated, including efforts to...

  • Develop new content pages,
  • Develop new worked example and homework problems,
  • Record video lectures and worked problem videos,
  • Develop interactive elements,
  • and finally proofread existing content

Everything from wanting to become an ongoing contributor to reporting a single error is welcome.

Additionally, the complete website for the Mechanics Map project is open source and available on GitHub at https://github.com/JacobMoore/mechanics_textbook. If you wish to use pieces of this website for your own open educational resources, feel free, just please honor the CC-BY-SA 4.0 license the work has been developed under. Details on this license can been found by clicking on the link in the footer.
